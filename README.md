## 🔒 背景
- 关注企业在敏捷项目思想方面的实践能力。

## 🔑 用途
- DevSecOps 是 Development、Security 和 Operations 的缩写，是软件生命周期内的敏捷项目管理的实践思想，在软件的生命周期的每个阶段（开发、测试、交付、转运维、灰度、生产等），让项目组拥有流水线式的持续作业能力。
